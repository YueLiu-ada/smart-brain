## `About this project`
* This project can make a face recognition on pictures that users upload. Users can sign in the existing accounts or register new accounts. This app can also count entries of each account.

### `Front-end`
* HTML5
* CSS3
* Javascript + React

### `Back-end`
* Node.js
* Express.js
* bcrypt-nodejs
* knex

### `Database`
* PostgreSQL

### `Deployment`
* Heroku
